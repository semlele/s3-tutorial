技術紹介

今回出題したピクトグラムはOpenPoseという技術を使用しています。

OpenPoseとは

身体に特殊な装置を取り付けることなく、AI(深層学習)を用いることで撮影動画及び画像から人物の骨格点（25点）のx，y座標を取得するシステム

出典：Zhe Cao， Gines Hidalgo， Tomas Simon， Shih-En Wei and Yaser Sheikh: OpenPose: Realtime Multi- Person 2D Pose Estimation using Part Affinity Fields， IEEE Transactions on Pattern Analysis and Machine Intelligence， 2019
